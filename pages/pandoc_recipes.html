pandoc recipes

by Jeremy Osborne, 2014.12.27 12:53:00

# A couple of recipes.
# html
# --output=FILE             output to file (file extension determines format)
# --write=html5             specify a specific output format to write to
# --standalone              standalone
# --smart                   smart (dafuq does that mean?)
# --highlight-style=STYLE   pygments (default), kate, monochrome, espresso, zenburn, haddock, tango
pandoc README.md --output=README.html --standalone --smart --write=html5 --highlight-style=tango

# pdf (long code blocks don't wrap)
pandoc README.md -o README.pdf --latex-engine xelatex --highlight-style tango

# odt (long code blocks don't wrap)
pandoc README.md -o README.odt --latex-engine xelatex --highlight-style tango
